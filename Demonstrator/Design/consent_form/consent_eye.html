Request for consent

Eye melanoma

Genecoop Principal Researcher asks for your consent to the following items:

Array Test

Purpose

An array test is needed to sequence your DNA from your saliva, and to produce a digital version of your DNA, which is needed for tumor classification in this research.

What does it mean for my personal data?

  • I will send a sample of my saliva for sequencing directly to Erasmus MC.
  • The array test will be performed by Erasmus MC, after which the sample will be stored at Erasmus MC only during the array test, after which it will be destroyed.
  • As a result of the array test my Raw DNA data file will be stored on Surf LabSafe and managed and protected by GeneCoop cooperation.
I agree to this

Copy Number Variation

Purpose

CNV is needed for tumor classification and personalised advice.

What does it mean for my personal data?

  • The analysis will be performed by Genecoop Prinicpal Investigator using Omnigen-GCP 
  • This analysis needs access to my Raw DNA data file stored on SURF LabSafe managed by Genecoop Cooperation.
  • I understand that as a result of the CNV analysis a CNV analysis report will be stored on SURF LabSafe managed and protected by Genecoop Cooperation
I agree to this
